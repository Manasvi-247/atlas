Problem Statement
🎯 Students move at different speeds, have different knowledge gaps, and learn
through different modalities. A fixed syllabus fails most students. The system
should generate and adapt the curriculum dynamically for each individual
learner.
What You're Building
🔨 An adaptive learning platform: intake assessment, AI-generated curriculum,
interactive lesson delivery, quiz-based adaptation, and a Socratic tutor chat.
Core Features
•
Knowledge Assessment – Intake quiz diagnosing current knowledge level across a
subject domain. Adaptive questioning that narrows to the boundary of what the learner
knows vs doesn't know.
•
Curriculum Generation – Generate a sequenced learning path from assessment results.
Structure into modules, lessons, and exercises with automatic prerequisite ordering.
•
Interactive Lessons – AI-generated lesson content with explanations, examples, and
analogies. Support text, code examples, and embedded practice questions inline.
•
Adaptive Progress – After each lesson or quiz, update the learner model. Revisit weak
areas, skip mastered content, and reorder upcoming lessons accordingly.
•
Socratic Tutor Chat – Chat interface for learner questions. Tutor guides toward the
answer using Socratic prompting rather than stating it directly.
•
Progress Dashboard – Visual progress map showing completed modules, mastery
scores, time spent, and recommended next steps.
